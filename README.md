# Visual Decision Support Mobile Robot

Система визуальной поддержки принятия решений для мобильной роботизированной платформы на основе компьютерного зрения, модели YOLOv8 и среды моделирования CoppeliaSim.

## Используемые технологии

- Python
- YOLOv8
- CoppeliaSim
- OpenCV
- NumPy
- ZMQ Remote API

## Структура проекта

### 01_Подготовка_данных
Подготовка и разделение набора данных.

### 02_Разметка_изображений
Проверка и обработка аннотаций изображений.

### 03_Обучение_YOLOv8
Обучение модели обнаружения объектов YOLOv8.

### 04_Оценка_модели
Оценка качества модели и анализ результатов.

### 05_Среда_CoppeliaSim
Работа со средой моделирования CoppeliaSim.

### 06_Сенсоры_и_камера
Получение данных с камеры Kinect и ультразвуковых датчиков.

### 07_Алгоритм_управления
Алгоритмы навигации и принятия решений.

### 08_Интеграция_YOLOv8
Интеграция компьютерного зрения с системой управления.

### 09_Полный_проект
Полная версия разработанной системы.
